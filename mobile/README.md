# Mobile App - Fluxwear

移动端电商应用，使用 Vue3 + Vant 开发。

## 开发预览

### 方法一：浏览器模拟手机预览（推荐用于开发）

1. 启动开发服务器：
```bash
npm run dev
```

2. 在浏览器中打开开发地址（通常是 `http://localhost:5173`）

3. 打开浏览器开发者工具：
   - **Chrome/Edge**: 按 `F12` 或 `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows)
   - **Safari**: 按 `Cmd+Option+I` (需要先在偏好设置中启用开发者菜单)

4. 切换到设备模拟模式：
   - 点击工具栏中的设备图标（📱）或按 `Cmd+Shift+M` (Mac) / `Ctrl+Shift+M` (Windows)
   - 选择设备型号（如 iPhone 14 Pro, iPhone SE, Pixel 5 等）
   - 可以自定义屏幕尺寸

5. 刷新页面即可看到移动端效果

### 方法二：真实手机预览

#### 步骤1：确保手机和电脑在同一WiFi
- 检查手机WiFi设置，确保连接的WiFi名称和电脑一样

#### 步骤2：获取电脑IP地址并启动服务

**Mac用户**：
```bash
# 在mobile目录下运行
cd mobile
npm run dev
# 然后在另一个终端窗口运行（查看IP地址）
ifconfig | grep "inet " | grep -v 127.0.0.1
```

**Windows用户**：
```bash
# 在mobile目录下运行
cd mobile
npm run dev
# 然后在命令提示符运行（查看IP地址）
ipconfig
# 找到"IPv4 地址"，通常是 192.168.x.x 格式
```

#### 步骤3：在手机上打开浏览器（详细操作）

   **iPhone (Safari浏览器)**：
   - 打开手机，找到并点击"Safari"图标（蓝色指南针图标）
   - 在浏览器底部，找到地址栏（显示网址的地方，通常在顶部或底部）
   - 点击地址栏，会弹出键盘
   - 输入：`http://192.168.1.11:5173`（将IP地址替换为你的实际IP）
   - 点击键盘上的"前往"或"Go"按钮
   
   **Android手机 (Chrome浏览器)**：
   - 打开手机，找到并点击"Chrome"图标（彩色圆形图标）
   - 在浏览器顶部，找到地址栏（显示网址的白色长条框）
   - 点击地址栏，会弹出键盘
   - 输入：`http://192.168.1.11:5173`（将IP地址替换为你的实际IP）
   - 点击键盘上的"前往"或回车按钮
   
   **如果找不到地址栏**：
   - 在浏览器中，向上滑动或向下滑动页面，地址栏通常会显示出来
   - 或者点击浏览器顶部的搜索框/地址框
   - iPhone Safari：地址栏在底部，点击即可输入
   - Android Chrome：地址栏在顶部，点击即可输入

#### 步骤4：输入网址
- 在地址栏中输入：`http://192.168.1.11:5173`
- **注意**：将 `192.168.1.11` 替换为你刚才查到的实际IP地址
- 输入完成后，点击"前往"或"Go"按钮

5. 如果无法访问，检查防火墙设置：
   - **Mac**: 系统设置 > 网络 > 防火墙，确保允许 Node.js 连接
   - **Windows**: 控制面板 > Windows Defender 防火墙，添加端口 5173 的例外

### 方法三：使用二维码（最方便，强烈推荐！）

#### 快速生成二维码（最简单）

1. **获取你的IP地址**（参考方法二的步骤2）

2. **生成二维码**：
   - 打开浏览器，访问：https://cli.im
   - 在输入框中输入：`http://[你的IP地址]:5173`
   - 例如：`http://192.168.1.11:5173`
   - 点击"生成二维码"
   - 二维码会显示在页面上

3. **用手机扫描**：
   - **iPhone**：打开相机App，对准电脑屏幕上的二维码，点击弹出的通知
   - **Android**：打开相机App或微信扫一扫，扫描二维码
   - 或者用微信扫一扫功能扫描

4. **如果扫描后无法打开**：
   - 检查手机和电脑是否在同一WiFi
   - 确认开发服务器正在运行（`npm run dev`）
   - 尝试直接在浏览器地址栏输入网址

## 注意事项

- 确保后端服务运行在 `http://localhost:8000`
- 如果使用真实手机预览，后端API代理可能无法正常工作，需要：
  - 将后端服务也配置为允许外部访问
  - 或者修改 API baseURL 为实际的后端地址

## 构建生产版本

```bash
npm run build
```

构建完成后，文件会在 `dist` 目录中。
