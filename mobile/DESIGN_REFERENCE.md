# 设计需求表达方法

## 方法一：使用设计工具绘制示意图

### 推荐工具：
1. **Figma** (https://figma.com) - 免费，在线使用
   - 可以精确绘制圆角、圆弧
   - 导出图片或分享链接

2. **Sketch** (Mac) - 专业设计工具

3. **Adobe Illustrator** - 矢量图工具

4. **在线工具**：
   - Excalidraw (https://excalidraw.com) - 简单手绘风格
   - Draw.io (https://draw.io) - 流程图工具，也可以画示意图

### 绘制要点：
- 标注圆角半径（如：24px）
- 标注凹入圆弧的半径和位置
- 标注对齐关系（价格标签与图片边缘对齐）

## 方法二：使用代码描述（SVG路径）

可以这样描述：
```svg
<svg>
  <!-- 图片区域：上、左、下左圆角，右下角有凹入圆弧 -->
  <path d="M 20,0 L calc(100% - 20),0 Q 100%,0 100%,20 L 100%,calc(100% - 24) Q 100%,100% calc(100% - 24),100% L 20,100% Q 0,100% 0,calc(100% - 20) L 0,20 Q 0,0 20,0 Z"/>
</svg>
```

## 方法三：使用数学描述

- **图片圆角**：上左、上右、下左 = 20px
- **右下角凹入圆弧**：
  - 圆心位置：图片右下角
  - 半径：24px（与价格标签圆角相同）
  - 方向：向内凹入（concave）
  - 角度：90度（右下象限）

## 方法四：使用CSS clip-path描述

```css
clip-path: polygon(
  0 0,                    /* 左上角 */
  100% 0,                 /* 右上角 */
  100% calc(100% - 24px), /* 右下角上方 */
  calc(100% - 24px) 100%, /* 右下角左侧 */
  0 100%                  /* 左下角 */
);
```

## 方法五：手绘草图 + 标注

1. 用手机拍照手绘草图
2. 在图片上标注：
   - 圆角半径
   - 凹入圆弧的半径和位置
   - 对齐关系

## 推荐工作流程

1. **快速验证**：先用 Excalidraw 画个简单示意图
2. **精确设计**：用 Figma 绘制精确尺寸
3. **标注说明**：在图上标注关键数值
4. **分享链接**：直接分享 Figma 链接或导出图片
